## for Developer

### How to use

Clone the repo via git:

```shell
$ cd path/to/root
$ git clone https://github.com/zce/chrome-markdown.git
```

And then install dependencies.

```shell
$ cd chrome-markdown
$ npm install
```

### Build it

```shell
$ gulp build
# archive
$ gulp archive
```

*Note: requires a node version >= 4 and an npm version >= 2.*
